01 Data Binding

Simple Binding

We can bind to simple values declared in JavaScript such as the value below. 

this.hello = 'world';
And we can bind to it in HTML using doubly curly braces 

{{main.hello}}
Which will produce 

{{main.hello}}

Challenge

Change the value this.hello in script.js to see the new value rendered.

Binding to an Object

We can also bind to properties of objects such as the object below. 

this.item = {
  name: 'My Name',
  description: 'This description'
};
And we can bind to it in HTML using doubly curly braces 

Name: {{main.item.name}}

Description: {{main.item.description}}
Which will produce 

Name: {{main.item.name}}

Description: {{main.item.description}}

Challenge

Change the value of a property on this.item in script.js to see the new value rendered.
